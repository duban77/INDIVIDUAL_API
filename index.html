API Utilizada: FakeStore API

¿Qué es FakeStore API?

FakeStore API es una API gratuita que proporciona productos falsos con información como nombre, precio, descripción, categoría e imagen. Es ideal para pruebas, prototipos y prácticas de consumo de APIs REST.
